# 部署流程

## 本地开发与公网访问

### 工作流程

1. **本地开发**：修改 `index.html`、`server.js` 等文件
2. **提交 Git**：完成阶段性开发后 commit
3. **同步到 Dify**：将修改后的文件复制到 nginx 目录和 Dify 服务器
4. **重启服务**：让改动生效

---

**Nginx 目录位置**：

### 文件服务器部署（server.js）

本地文件服务器 `server.js` 保存用户上传的图片：

- **启动命令**：`node server.js`
- **端口**：3001
- **上传目录**：`c:/Users/zhang/Desktop/潮墨智能体/uploads/`

**公网访问方案**：通过 nginx 代理，无需额外暴露 cpolar 端口

1. 把 `server.js` 和 `uploads/` 目录复制到 Dify 服务器
2. 在 Dify 服务器上启动 `node server.js`
3. 修改 Nginx 配置添加代理（见下方）
4. 重启 nginx

**Nginx 配置**（在 `default.conf.template` 中添加）：
```nginx
# 注意：Docker 容器内 localhost 指向容器自己，需要用宿主机 IP
location /agent/upload {
    proxy_pass http://192.168.71.11:3001/upload;
    include proxy.conf;
}

location /agent/uploads {
    proxy_pass http://192.168.71.11:3001/uploads;
    include proxy.conf;
}
```

**重要**：如果宿主机 IP 变化，需要更新这里的 IP 地址。

### 关键提醒

1. 每次修改 index.html 后都要复制到 nginx 目录并重启
2. 文件服务器需要手动启动，或设置开机自启
3. Dify 工作流配置在 Dify 服务器上，不在这个仓库
4. cpolar 免费版只能暴露 80/443 端口，文件上传通过 nginx 代理到 server.js 解决

## 开机启动步骤

电脑重启后，按以下顺序启动所有服务：

### 1. 启动 Docker Desktop

在 Windows 搜索栏搜索 **Docker Desktop**，打开等待其完全启动（约 1-2 分钟）。

验证启动成功：
```bash
docker ps
```

### 2. 启动 Dify 容器

**一般情况下不需要手动启动**，Docker 打开后会自动启动之前部署的 Dify 容器。

只有**首次部署或容器被手动停止**时，才需要：
```bash
cd C:/Users/zhang/Downloads/dify-main/docker
docker compose up -d
```

验证容器运行状态：
```bash
docker ps
```

应看到 `api`, `web`, `nginx`, `db`, `redis` 等容器在运行。

### 3. 启动 cpolar 内网穿透

在 cpolar 安装目录下打开终端，执行：
```bash
cd C:/Users/zhang/AppData/Local/cpolar
cpolar.exe http 80
```

或者如果已经配置了 authtoken，可以用：
```bash
cpolar.exe start-all
```

成功后显示：
```
Session Status  online
Forwarding      http://1b693889.r39.cpolar.top -> localhost:80
```

### 4. 启动文件服务器 server.js

新建终端窗口：
```bash
cd c:/Users/zhang/Desktop/潮墨智能体
node server.js
```

成功后显示：
```
文件服务器已启动: http://localhost:3001
上传目录: c:/Users/zhang/Desktop/潮墨智能体/uploads
```

### 5. 验证服务

访问以下地址确认服务正常：
- 主站：http://1b693889.r39.cpolar.top/agent/
- Dify API：http://1b693889.r39.cpolar.top/agent/v1/info

### 6. 同步最新代码到 Dify（如有修改）

如果 `index.html` 有更新：
```bash
cp "c:/Users/zhang/Desktop/潮墨智能体/index.html" "C:/Users/zhang/Downloads/dify-main/docker/volumes/certbot/www/index.html"
docker restart docker-nginx-1
```
